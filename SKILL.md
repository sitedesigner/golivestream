# GoLivestream — Reveting Content Engine

A complete operational playbook for running a weekly LinkedIn Live show as a B2B content and lead generation engine. Based on Jessie Lizak's Reveting methodology.

## The 5 Skills

| # | Skill | Purpose |
|---|-------|---------|
| 1 | `reveting-show-setup` | Master variable intake — Tier 1 (show constants) + Tier 2 (episode variables) |
| 2 | `reveting-email-flows` | 6-sequence email system — guest outreach through ongoing nurture |
| 3 | `reveting-calendar-flows` | Calendar architecture — booking pages, reminders, no-show recovery |
| 4 | `reveting-ghl-workflows` | Go High Level automation — pipelines, workflows, contact schema |
| 5 | `reveting-streamyard-flows` | StreamYard production — scenes, overlays, run-of-show, recording |

## Quick Start

1. Start with `reveting-show-setup` — complete Tier 1 for your show
2. Use `reveting-email-flows` to build your guest outreach sequences
3. Configure `reveting-calendar-flows` for booking and reminders
4. Build `reveting-ghl-workflows` for CRM automation
5. Run `reveting-streamyard-flows` for live production
